What is FlowMCP?

FlowMCP is an open-source framework for turning existing web APIs into a standardized Model Context Protocol (MCP) interface. It allows AI assistants and applications to access web API data and services in a structured, consistent manner.

By acting as a ‚Äúuniversal adapter‚Äù between AI and external APIs, FlowMCP makes it easier to bring real-world data (especially from Web3/blockchain services) into AI-driven tools.

Key Features

  • Declarative schemas: APIs are defined in modular files describing endpoints, parameters, and expected results.
  • Pre/Post processors: Normalize data, format responses, or modify inputs as needed.
  • Test coverage: Each schema includes automated test cases to verify functionality.
  • AI-friendly text output: Structured and natural-language responses are returned for easy LLM integration.

Usage & Schema Library

Run a local or remote MCP server and load schemas from the official schema repository. The server exposes API routes as named ‚Äútools‚Äù an AI can invoke.

Over 200+ tools are available out of the box, covering CoinGecko, Chainlink, Moralis, Etherscan and more.
