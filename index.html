What is FlowMCP?

FlowMCP is an open-source framework for exposing REST APIs through a consistent Model Context Protocol (MCP) interface, optimized for AI agents and assistants.

Key Features

  • Declarative schemas: Define APIs without custom code.
  • Pre/Post processing: Normalize and customize request/response data.
  • Built-in testing: Each tool includes test cases.
  • Text output: All responses are optimized for AI-readable text.

Usage

Run a local or remote MCP server and load schema modules to expose APIs as tools for AI clients.
